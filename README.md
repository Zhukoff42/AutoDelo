🕵️‍♂️ Auto-Delo - Автоматическое расследование дел для копов!
Auto-Delo — это удобный скрипт на MoonLoader для сотрудников МЮ, который полностью автоматизирует процесс расследования убийств на сервере. Больше никаких рутинных отыгровок вручную и долгого переписывания улик! 📜🚀

✨ Основные возможности:
Автоматическое РП: Скрипт сам отыграет за вас все необходимые действия (/me, /do, /todo) при начале расследования и заполнении бланков. 🎭

Умный сбор улик: Скрипт автоматически парсит и сохраняет точную дату, время и орудие убийства прямо из чата или диалогов. 🔍

Быстрая вставка в CEF: Наведение курсора и вставка собранных данных в CEF-окна в один клик с помощью горячих клавиш. ⚡

⚙️ Установка:
Убедитесь, что у вас установлен MoonLoader.

Переместите файл AutoDelo.lua в папку `moonloader`.

Заходите в игру и используйте команду `/setdelo` для открытия меню настроек.

📝 Управление:
`/delo` — начать процесс расследования или остановить его.

`/setdelo` — открыть/закрыть меню настроек скрипта.

`ALT + 1` — вставить сохраненную дату и время убийства в поле ввода CEF.

`ALT + 2` — вставить сохраненное орудие убийства в поле ввода CEF.

`F12` — экстренная остановка работы скрипта.
