🔎 Auto-Delo - Автоматизация расследования убийств для копов
Описание:
Надоело каждый раз вручную собирать улики, запоминать время и судорожно перепечатывать названия оружия в окна? 🥱 Скрипт AutoDelo сделает всю рутину за вас! 
Он разработан специально для того, чтобы максимально облегчить и ускорить процесс расследования дел.

Основные фичи:

⚙️ Полная автоматизация: Скрипт сам начинает осмотр места убийства, взаимодействует с уликами и запускает процесс.

📝 Умный сбор данных: Автоматически вытаскивает из чата и диалогов точную дату, время и тип оружия.

🎭 Встроенное RP: Красивые и плавные отыгровки через /me, /do и /todo во время работы (при желании можно отключить в меню).

🕹️ Управление и команды:

/delo — запустить процесс расследования на месте убийства (или остановить его).

/setdelo — открыть компактное меню настроек (включение/отключение отыгровок).

ALT + 1 — вставить сохраненную дату и время в поле ввода CEF.

ALT + 2 — вставить сохраненное орудие убийства в поле ввода CEF.

F12 — экстренная остановка скрипта, если что-то пошло не так.
